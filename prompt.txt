📊 REQUERIMIENTO: GENERACIÓN, AJUSTE DE TAMAÑO E IMPRESIÓN DE CÓDIGO QRContexto del requerimiento:Cada niño en KidLink-App debe tener la opción de mostrar un código QR permanente que vincule a su misma URL de emergencia pública ($WEB_URGENCIA_URL/nino/$id_tag). Necesito agregar una sección dentro de la gestión del niño que permita visualizar este QR, modificar su tamaño dinámicamente mediante un control deslizante y prepararlo para impresión o descarga.Instrucciones de desarrollo para el agente:Librería de Renderizado QR:Añade el paquete estándar qr_flutter en el archivo pubspec.yaml para generar los códigos QR de forma nativa como widgets de imagen de alta definición, evitando el uso de APIs externas de internet que requieran conexión.Interfaz de Visualización del QR:Diseña una vista limpia centrada en el código QR del niño.Añade un control deslizante (Slider) en la parte inferior de la pantalla. Al mover este control, el widget del código QR debe cambiar de tamaño en tiempo real (por ejemplo, permitiendo escalar entre un tamaño mínimo de 100x100 píxeles para etiquetas pequeñas, hasta 300x300 píxeles para hojas completas).Funcionalidad de Exportación e Impresión:Añade un botón destacado de "Imprimir / Compartir QR".Utiliza el paquete pdf o las funciones nativas de la hoja de compartir del sistema (share_plus) para empaquetar el código QR generado en el tamaño seleccionado como un archivo de imagen limpia (PNG) o un documento listo para mandar a una impresora hogareña o de stickers.Asegúrate de incluir un texto sutil debajo del QR impreso que diga: "En caso de emergencia, escanee este código o acerque el celular para leer el chip NFC" para dar instrucciones claras a quien lo encuentre.
